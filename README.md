Java code example of  design creational pattern pool object.

Example to apply good practise in software developmemnt: test and mesurement.

This proyect is a fork of [clopezno/poolobject](https://github.com/clopezno/poolobject) to solve the exercise "Medir para caracterizar entidades de productos y procesos
software".
